## Ключевые концепции

- **Блуждающий нерв (вагус)** — главный компонент ПНС, ~80% волокон — афферентные
- **Холинергический противовоспалительный путь** — нейроиммунный механизм контроля воспаления
- **α7-никотиновые рецепторы** — ключевые рецепторы на макрофагах для противовоспалительного эффекта
- **Вариабельность сердечного ритма (HRV)** — неинвазивный маркер вагального тонуса

## Заключение

Парасимпатическая нервная система — не просто "тормоз" для симпатической активации. Через ацетилхолин и холинергический противовоспалительный путь она:

1. **Регулирует иммунный ответ** — предотвращает чрезмерное воспаление
2. **Связывает мозг и иммунитет** — обеспечивает нейроиммунную коммуникацию
3. **Влияет на метаболизм** — через ось кишечник-мозг
4. **Определяет устойчивость к стрессу** — высокий вагальный тонус = лучшая адаптация

Понимание этих механизмов открывает пути к:
- Новым методам лечения воспалительных и аутоиммунных заболеваний
- Обоснованному применению практик активации ПНС
- Профилактике хронических заболеваний через модуляцию вагального тонуса
