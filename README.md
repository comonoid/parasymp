# Парасимпатическая нервная система

Комплексное руководство по парасимпатической нервной системе: теоретические основы, связь с иммунитетом и практические методы активации.

## О проекте

Парасимпатическая нервная система (ПНС) отвечает за состояние "отдыха и переваривания" — замедляет сердцебиение, снижает артериальное давление, стимулирует пищеварение и способствует восстановлению организма. Через холинергический противовоспалительный путь ПНС также регулирует иммунный ответ и контролирует воспаление.

## Ключевые концепции

- **Блуждающий нерв (вагус)** — X пара черепных нервов, главный компонент ПНС, иннервирует большинство внутренних органов
- **Вагальный тонус** — показатель активности блуждающего нерва, измеряется через вариабельность сердечного ритма (HRV)
- **Холинергический противовоспалительный путь** — механизм, через который вагус контролирует системное воспаление
- **α7-никотиновые рецепторы** — рецепторы на макрофагах, активация которых подавляет выброс провоспалительных цитокинов

## Быстрый старт

Для начала рекомендуется:

1. Ознакомиться с [общими сведениями о ПНС](description/01_overview.md)
2. Изучить [дыхательные техники](01_breathing.md) — самый доступный метод активации
3. Попробовать [когерентное дыхание](01_breathing.md) — 5-6 вдохов в минуту в течение 5-10 минут
4. Измерить свой [HRV](description/07_hrv.md) для отслеживания прогресса
